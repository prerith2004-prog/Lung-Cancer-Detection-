# Lung Cancer Detection Using Ensemble Algorithms

## Project Overview

This project implements a binary classification system for detecting
Malignant and Benign lung cancer cases using ensemble machine learning techniques.

## Algorithms Used

1. Random Forest
2. Gradient Boosting
3. XGBoost
4. Voting Ensemble
5. Stacking Ensemble

## Features

- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Feature Selection
- Ensemble Learning
- ROC-AUC Analysis
- Threshold Optimization

## Technologies

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-Learn
